Null Hypothesis (H) vs. Alternative Hypothesis (H)
Null Hypothesis (H): The default assumption, often representing "no effect" or "no difference."
Alternative Hypothesis (H): The hypothesis the researcher aims to support, suggesting a significant effect or difference.
Type I Error (False Positive): Rejecting the null hypothesis when it is actually true. Denoted by: (Alpha) Type II Error (False Negative): Failing to reject the null hypothesis when the alternative hypothesis is true. Denoted by: (Beta)
